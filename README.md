## üéØ Learning Objectives
Through this project, I am actively learning and applying:
* **SwiftUI:** Building declarative, responsive, and modern user interfaces.
* **Architecture:** Implementing a clean MVVM (Model-View-ViewModel) pattern to separate logic from UI.
* **Project Organization:** Structuring an Xcode project by feature (Features, Core, Models, Services) rather than just dumping files in one place.
* **Apple Frameworks:** Exploring tools like `MapKit` for routing and `CoreLocation` for tracking user coordinates.
